# Handheld Calculator Using Keypad

## Overview
This project is a handheld calculator implemented on STM32. It uses a keypad for user input and an LCD to display numbers and calculation results. The project focuses on embedded input handling, processing logic, and display control.

## Features
- Keypad-based number and operator input.
- Basic arithmetic operations
- LCD display for input and result output
- Embedded firmware implementation on STM32

## Hardware / Platform
- STM32 development board
- Matrix keypad
- LCD display
- Power supply for embedded operation

## Software / Tools
- Embedded C
- Keil uVision
- STM32CubeMX
- Serial and functional testing

## How It Works
The system scans keypad input, identifies numbers and operators, processes arithmetic operations, and updates the LCD with the corresponding result. The firmware handles input reading, operation management, and display output in real time.

## Repository Structure
- `main.c` or source files: main application logic
- driver files: keypad/LCD handling
- `README.md`: project description
